# 网格交易系统

这是一个基于 Binance API 的加密货币网格交易系统。

## 主要功能

- ✅ 支持 BTC/USDT 和 ETH/USDT 交易对
- ✅ 实时价格显示和 K 线图表
- ✅ 自定义网格参数
- ✅ 支持现货和合约交易
- ✅ 本地 API 密钥存储，安全可靠

## 系统要求

- Windows 7/8/10/11
- 4GB RAM 以上
- 500MB 可用磁盘空间

## 下载安装

访问 [下载页面](https://tiance666.github.io/grid-trading/) 获取最新版本。

## 安装说明

1. 下载安装程序
2. 双击运行安装程序
3. 按照安装向导完成安装
4. 使用桌面快捷方式启动程序

## 开发说明

本项目使用以下技术栈：

- 后端：Python + FastAPI
- 前端：HTML + JavaScript
- 打包：Electron
