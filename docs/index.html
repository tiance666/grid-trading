网格交易系统

项目简介

这是一个基于 Binance API 的加密货币网格交易系统。系统采用 FastAPI 构建，提供直观的 Web 界面，支持灵活的网格交易策略配置。

主要特点

  • 自动网格交易：系统自动在设定价格区间内进行买卖操作
  • 多币种支持：目前支持 BTC/USDT、ETH/USDT 交易对
  • 实时价格监控：通过 WebSocket 实时更新市场价格
  • 灵活配置：支持自定义网格数量、网格高度等参数
  • 安全可靠：通过 API Key 认证，保障账户安全

使用指南

系统要求

  • Python 3.8 或更高版本
  • Binance API Key 和 Secret Key

快速开始

  1. 克隆项目代码：
    git clone https://github.com/tiance666/grid-trading.git
  2. 安装依赖：
    pip install -r requirements.txt
  3. 启动服务器：
    cd src/server && uvicorn app:app --host 0.0.0.0 --port 8000
注意： 在开始交易前，请确保：
  • 已经理解网格交易的原理和风险
  • 使用正确的 API 权限（仅需要交易权限）
  • 从小额开始测试系统功能

配置说明

网格参数

  • 网格数量： 建议设置4-50个网格
  • 网格高度： 建议范围0.1%-10%，根据市场波动调整
  • 投资金额： 根据个人风险承受能力设置
